Developer RoadMap

  • Step1
  • Programming Basic1 .2023 year
  • 웹에 대한 기본적인 이해
  • 웹페이지 기본언어 이해(HTML, CSS, JS)
  • 프로그래밍 언어에 대한 기본문법을 익히기(JAVA)
  • 데이터베이스 기술이해하기(Oracle, Mysql)
  • 코드 및 버전 관리, 개발자 협업을 위한 Git, Github 이해하기
  • Step2
  • Programming Basic2 .2023 year
  • Framework이해하기(Spring Boot)
  • 읽기좋은 코드를 구현하는 것, 코드를 개선해 변경해보는 경험하기
  • 팀원들과 협업을 통해 프로젝트 진행하고, 피드백을 받아 지속적으로 개선하기
  • 학습한 내용들을 프로젝트에 적용하기
  • Step3
  • Programming Basic3 .2023 year
  • 배운 기술 스택으로 포트폴리오 제작
  • 이력서 작성 및 기술면접준비
  • 신입 백엔드 개발자 직업 지원을 시작
  • 네트워킹, 적극적으로 인턴십 추구
  • 주니어 개발자 직위 또는 컨설팅은 백엔드 개발자로서의 경력을 시작하고 발전
  • Step4
  • Programming Develop .2024 year
  • API(응용 프로그래밍 인터페이스) 개발 및 관리
  • 정보처리기사 및 관련자격증 취득하기
  • Java, Python, JavaScript(Node.js) 심층적 이해 확장
  • 코드 관련 문제에 대한 개발 솔루션을 작성하고, 테스트하고, 유지하는 것
  • 새로운 기능을 식별하려면 개발자, 디자이너 및 시스템 관리자와 효과적으로 커뮤니케이션
  • Step5
  • Programming Develop .2025 year ~
  • 시스템의 어려움에 대한 해결책을 제공
  • 개념에 대한 이해를 확고히 하는 방법을 배우는 동안 많은 프로젝트를 만드는 것
  • 기술을 향상시키고 빠르게 진행되는 업계 기술 발전에 대비하려면 지속적인 학습 태도를 가지기
